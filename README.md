# Transjux Web Extension

Safari Web Extension source code for Transjux.

## Included Files

- `manifest.json`
- `api.js`
- `background.js`
- `content.js`
- `popup.html`
- `popup.js`
- `options.html`
- `options.js`
- `icons/`

## Shortcuts

- `Option + T`: Translate selection
- `Option + Shift + T`: Translate current page
- `Option + I`: Toggle immersive mode

Shortcut binding still depends on Safari extension shortcut settings.
