# Transjux Web Extension

This directory contains the Safari Web Extension source code.

## Files

- `manifest.json`
- `background.js`
- `content.js`
- `popup.html`
- `popup.js`
- `options.html`
- `options.js`
- `api.js`
- `icons/`

## Shortcuts (suggested)

- `Option + T`: Translate selection
- `Option + Shift + T`: Translate current page
- `Option + I`: Toggle immersive mode

Shortcut behavior depends on Safari extension shortcut settings.
